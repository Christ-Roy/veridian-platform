2. Conformité du Site Web
Stripe va scanner votre site (ou demander à un humain de le vérifier). Pour ne pas être refusé, votre site doit comporter :
HTTPS obligatoire : Votre site doit être sécurisé (certificat SSL).
CGV / CGU : Des Conditions Générales de Vente claires.
Politique de remboursement : Elle doit être explicitement mentionnée.
Mentions légales : Nom de l'entreprise, adresse, contact (email/téléphone).
Description claire : Ce que vous vendez et à quel prix (la devise doit être visible).
Logos de paiement : Idéalement, afficher les logos Visa/Mastercard au moment du paiement.
